# Flutter Navigator 2.0 Labs (go_router)

This repository contains three lab assignments demonstrating navigation in Flutter using the `go_router` package.

## Labs Included:

### 1. Lab 1: Basic Navigation
* Simple navigation between a Home screen and a Details screen.
* Demonstrates `GoRouter` configuration and `context.go()`.

### 2. Lab 2: Path & Query Parameters
* Demonstrates how to pass data through the URL.
* Uses `:id` as a path parameter and `filter` as a query parameter.

### 3. Lab 3: Navigation Stack (Bonus)
* Adds a "Back to Home" button on the product page.
* Demonstrates nested layouts using `Column` and returning to the root route.
